float MCP3421GetValue ( void  )

This funtion gets the value from the ADC. The funtion as blocking properties, if the convertion is not completed the device is continuous asked if the convertion is completed. When is completed the value in volts is returned.

Returns
The voltage on the terminals of the ADC.
